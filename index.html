All of the previous images were exported in Photoshop; but you also have the option of exporting images from Sketch (@1x , @2x etc.):

Important takeaways:

  1. Sketch isn't built to process images, so the quality of its image exports isn't as good as Photoshop, even with the same number of pixels. Take note of the file sizes too! Overall, this isn't a competitive practice for web design.

One optimization for Sketch exports is rounding assets to the nearest pixel (Arrange > Round to Nearest Pixel):

Important takeaways:

  1. The impacts of rounding raster graphics varies, but is still quite minimal, and will not compete with retina image quality; and,
  2. If you round vector graphics (ex. icons) to the nearest pixel, you'll break them. Therefore, it is NOT suggested to select an entire Sketch file and round.
